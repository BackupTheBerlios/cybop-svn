Best Procedure:
- insert bitmap graphic into OpenOffice presentation
- export graphic (just this one page of the presentation) in greyscale colour into EPS file
- open EPS file in kpdf
- print graphic to PDF file using PDF printer

Alternative:
- insert bitmap graphic into OpenOffice presentation
- print graphic (just this one page of the presentation) in greyscale colour into PS file
- open PS file in kghostview
- print graphic to PDF file using PDF printer
