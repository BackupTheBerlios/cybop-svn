How to upload files to our website using Windows OS?

 

 

For those developers who use Windows I recommend WinSCP to upload files to the Resmedicinae Website. It is not possible to use ftp, because it is not secure enough. With WinSCP you can easily transfer your websites, screenshots or other files to the Sourceforge sever.
