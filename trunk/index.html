CYBOP stands for Cybernetics Oriented Programming and is an Object- oriented, Component- oriented, Ontology- based Framework for software development that makes use of new design concepts and ideas which are basically taken from nature.

CYBOP's architecture is platform- independent and programming- language- neutral, although it is currently implemented in Java code.

CYBOP has its roots in the ResMedLib Framework of the Res Medicinae project which aims to provide users with a free software solution for medicine.
