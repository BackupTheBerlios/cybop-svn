Cybernetics Oriented Language (CYBOL)

CYBOL is the result of many years of (theoretical and practical) research in software development, striping fields such as:

  • Structured/ Procedural Programming
  • Object-Oriented Programming
  • Software Architecture/ Design Patterns
  • Component-Based Design (Lifecycle, Concern, Aspect)
  • Ontology-Structured Domain/ Knowledge
  • Document/ User Interface Markup
  • Persistence Mechanisms
  • System Communication

Syntax
  • Extensible Markup Language (XML)
Semantics
  • Hierarchy
  • Statics (structure, struct/ record/ class)
  • Dynamics (process, operation/ function/ procedure/ workflow)
  • Translators
  • Inheritance (super/ sub)
  • Instantiation (parent/ child)
  • Meta information about children

Before the end of 2004, this page will contain a complete specification of CYBOL.
It will also show and explain the disadvantages of current programming languages.
CYBOL will be easy to write, also by non-programmers.
