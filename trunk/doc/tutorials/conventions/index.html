Code Conventions

These style guide rules can be applied to C as well as XML code.
They closely adhere to the Java Programming Language code convention.

Important issues

1 Avoid using beautifier (code formatting) tools.
2 Use spaces and never tab characters.
3 Indent code by 4 spaces. (Editors offer to set the TAB key to 4 spaces)
4 Put the opening brackets to the end of a line and not on an extra line.
5 Make use of spaces and white lines to separate code logically.
6 Comment all code exhaustively.
