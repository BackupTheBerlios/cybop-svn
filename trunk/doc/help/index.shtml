How to help

Tasks

Any kind of help is always very welcome! Help doesn't necessarily mean Programming. There is plenty of ways you can contribute to our work, for example:

  • Participate in our mainling lists
  • Test, play with and (one day in the future) use our software
  • Report bugs and share your knowledge to bring forward our requirements (feature) analysis
  • Develop source code
  • Design graphics (icons etc.)
  • Maintain and update our homepage
  • Translate texts into your native language
  • Moderate a mailing list
  • Coordinate the release plans
  • Promote and advertise our project

For more concrete development tasks, read the project plan!

Support

The software of this project is free in use and also its source code is. The whole project is run on resources and funds that are supplied lead by pure idealism. If you would like to support our work financially or by supplying materials, please use this information.
