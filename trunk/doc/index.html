Cybernetics Oriented Programming (CYBOP)

CYBOP stands for Cybernetics Oriented Programming and is a new theory
for software development, building on concepts which are taken from nature.

CYBOL is an XML-based application programming- and knowledge
specification language and thus completely platform-independent.

CYBOI is the corresponding interpreter that is needed to execute
systems defined in CYBOL.
